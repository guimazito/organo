## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/guimazito/organo.git
   ```
2. Enter in organo repository
   ```sh
   cd organo
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
